📖 Professional Voice Chess Notation

🎯 Basic Move Format

Standard format: [Piece] [File] [Rank]

  • e4 → Say: "e four" or "pawn e four"
  • Nf3 → Say: "knight f three"
  • Bc4 → Say: "bishop c four"
  • Qd4 → Say: "queen d four"
  • Ra1 → Say: "rook a one"

⚔️ Captures

Format: [Piece] takes [Square]

  • Bxc4 → Say: "bishop takes c four"
  • exd5 → Say: "e takes d five" or "pawn takes d five"
  • Nxe5 → Say: "knight takes e five"

Alternative: Can also say "bishop captures c four"

🏰 Castling

  • O-O → Say: "castle kingside" or "short castle"
  • O-O-O → Say: "castle queenside" or "long castle"

👑 Pawn Promotion

Format: [Move] promote [Piece]

  • e8=Q → Say: "e eight promote queen"
  • a1=N → Say: "a one promote knight"

Alternative: "e eight queen" or "promote to queen"

🔤 Piece Names

  • King = K
  • Queen = Q
  • Rook = R
  • Bishop = B
  • Knight = N (say "knight" NOT "N")
  • Pawn = (no letter, just the square)

📍 Files and Ranks

Files (columns): a, b, c, d, e, f, g, h

Ranks (rows): 1, 2, 3, 4, 5, 6, 7, 8

Say them clearly:

  • "a" = "ay" (like the letter A)
  • "b" = "bee"
  • "c" = "see"
  • "d" = "dee"
  • "e" = "ee"
  • "f" = "eff"
  • "g" = "gee"
  • "h" = "aitch" or "haitch"

💡 All Valid Ways to Say Moves

Example: Moving pawn to e4

  • ✅ "e four"
  • ✅ "pawn e four"
  • ✅ "pawn to e four"
  • ✅ "e 4" (with pause)

Example: Knight to f3

  • ✅ "knight f three"
  • ✅ "knight to f three"
  • ✅ "n f three"

Example: Bishop captures c4

  • ✅ "bishop takes c four"
  • ✅ "bishop captures c four"
  • ✅ "b takes c four"
  • ✅ "take c four with bishop"

🎤 Tips for Best Recognition

  • Speak clearly and naturally (not too fast)
  • Pause slightly between file and rank: "e... four"
  • English voice recognition is 85-90% accurate (vs 70% Spanish)
  • System finds closest legal move automatically
  • If wrong move, use "Undo" button
  • Can mix notation styles (the system understands all)
