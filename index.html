üî§ Piece Names

  • King = K

  • Rook = R

üìç Files and Ranks

Files (columns): a, b, c, d, e, f, g, h

Ranks (rows): 1, 2, 3, 4, 5, 6, 7, 8

Say them clearly:

  • "a" = "ay" (like the letter A)
  • "b" = "bee"
  • "c" = "see"
  • "d" = "dee"
  • "e" = "ee"
  • "f" = "eff"
  • "g" = "gee"
  • "h" = "aitch" or "haitch"
