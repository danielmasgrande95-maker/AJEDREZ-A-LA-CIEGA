📖 Sistema Fonético Mejorado

🔤 Alfabeto Fonético (sonidos diferenciables)

  • a → Di: "a"
  • b → Di: "bi"
  • c → Di: "co"
  • d → Di: "du"
  • e → Di: "e"
  • f → Di: "fa"
  • g → Di: "ge"
  • h → Di: "ju"

🎯 Formato de Movimiento

Estructura: [letra]-[número] hasta [letra]-[número]

Ejemplos:

  • e2 → e4 = "e-2 hasta e-4"
  • h4 → f2 = "ju-4 hasta fa-2"
  • b1 → c3 = "bi-1 hasta co-3"
  • g8 → f6 = "ge-8 hasta fa-6"

🏰 Enroques

  • Enroque corto: "enroque corto"
  • Enroque largo: "enroque largo"

💡 Tips

  • Pronuncia claramente el guion entre letra y número
  • Pausa después de "hasta"
  • Ejemplo completo: "bi UNO hasta co TRES"
  • Si falla, usa el teclado (100% confiable)
