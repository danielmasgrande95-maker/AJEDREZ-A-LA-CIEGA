📖 Cómo Jugar al Ajedrez a Ciegas

🎯 Objetivo del Juego

El ajedrez a ciegas es una modalidad donde juegas sin ver el tablero (o con opción de ocultarlo). Debes memorizar las posiciones y decir tus movimientos por voz usando un alfabeto especial.

🔤 Alfabeto Fonético Español

Para que el reconocimiento de voz funcione perfectamente, debes usar estas palabras en lugar de las letras:

  • A = "Ambrosio"
  • B = "Braulio"
  • C = "Carlitos"
  • D = "Danielín"
  • E = "Emilio"
  • F = "Facun"
  • G = "Golfo"
  • H = "Harry"

📝 Ejemplos de Movimientos

Peones:

  • e4 → Di: "Emilio cuatro"
  • d5 → Di: "Danielín cinco"
  • Captura: exd5 → Di: "Emilio por Danielín cinco"

Piezas:

  • Nf3 (Caballo) → Di: "Caballo Facun tres"
  • Bc4 (Alfil) → Di: "Alfil Carlitos cuatro"
  • Bxc4 (Captura) → Di: "Alfil por Carlitos cuatro"
  • Qh5 (Dama) → Di: "Dama Harry cinco"
  • Rd1 (Torre) → Di: "Torre Danielín uno"

Movimientos Especiales:

  • Enroque corto (O-O) → Di: "enroque corto"
  • Enroque largo (O-O-O) → Di: "enroque largo"
  • Promoción → Di: "Peón Emilio ocho corona a dama"

🔊 Sonidos de Confirmación

  • ✅ Pitido AGUDO (ti-lín) = Movimiento correcto
  • ❌ Pitido GRAVE (tum-tum) = Error o movimiento ilegal

🎮 Controles

  • Activar Voz: Inicia el reconocimiento de voz. El micrófono se queda activo hasta que hables.
  • Repetir: Repite el último mensaje hablado.
  • Ocultar/Mostrar: Esconde el tablero para jugar 100% a ciegas.
  • Deshacer: Retrocede tu último movimiento y el de la computadora.
  • Nueva Partida: Reinicia el juego.

💡 Consejos

  • Habla claro y despacio
  • Di la palabra completa: "Ambrosio", no "a"
  • Espera a que la computadora termine de hablar
  • Tómate tu tiempo para pensar - el micrófono esperará
  • Si el reconocimiento falla, prueba escribir el movimiento manualmente
  • En móvil Android: asegúrate de dar permiso al micrófono

⌨️ Atajos de Teclado

  • N = Nueva partida
  • M = Mostrar/Ocultar tablero
  • E = Activar voz
  • R = Repetir locución
  • H = Ayuda (este menú)
